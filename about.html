Welcome to AnimeDB

AnimeDB is a web-based platform built to manage and track your anime titles. Keep a record of anime you've watched, are currently watching, or plan to watch. Manage details like genre, ratings, reviews, and more!

Project Overview

This project helps users organize their anime collection in a beautiful, easy-to-use interface. With CRUD (Create, Read, Update, Delete) operations and a dynamic database, AnimeDB is a powerful tool for any anime fan.

Key Features

  • Store and manage your favorite anime.
  • Track your progress and status (Watching, Completed, etc.).
  • Sort by genre, release year, or ratings.
  • Leave reviews and feedback for each anime.
  • Responsive design for all devices.

Technical Details

AnimeDB is built using PHP and PDO for secure and efficient database management. The system uses modern web technologies to provide a seamless experience for users, allowing them to manage their anime data easily and securely.
